## Installation on your computer

    - [Install for Windows](https://git-scm.com/downloads/win)

    - [Install for Mac](https://git-scm.com/downloads/mac)

    - [Install for Linux](https://git-scm.com/downloads/linux)

Verify installation in your terminal

    ```bash
    git --version
    ```

## Login in Git

    ```bash
    git config --global user.name { your_name }
    git config --global user.email { your_email }
    ```
